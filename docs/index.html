Wat is progressive enhancement??

Progressive enhancement is het concept waarbij je website werkt op de meest fundamentele en simpele webtechniek (HTML) en vervolgens voeg je steeds een nieuwe "laag" toe, de CSS en Javascript. Door progressive enhancement kan je website gebruikt worden door zoveel mogelijk gebruikers maar biedt het ook nieuwere browsers een betere ervaring.
